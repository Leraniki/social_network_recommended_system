# Social Network Recommendation System

Сервис рекомендаций постов для социальной сети.  
Проект реализован в рамках обучения на Karpov.Courses (StartML).

## Задача
Разработать сервис, который для каждого пользователя возвращает топ-5 постов, наиболее релевантных его интересам в данный момент времени.

## Стек технологий
*   **Python 3.10**
*   **FastAPI** — для реализации REST API сервиса.
*   **CatBoost** — для ранжирования кандидатов (Ranking).
*   **PostgreSQL** — база данных признаков и текстов.
*   **Pandas** — для обработки данных.

## Архитектура решения

Рекомендательная система построена по двухуровневой архитектуре (хотя в коде для упрощения Item-Based реализован как фича):

1.  **Этап 1 (Feature Engineering & Retrieval Context):** 
    *   Рассчитывается **Item-Based** схожесть постов на основе со-встречаемости лайков (Co-occurrence matrix).
    *   Создается признак `item_similarity_score`, показывающий, насколько конкретный пост похож на те, что пользователь лайкал ранее.
    *   Признаки пользователей и постов предварительно загружены в SQL-базу для быстрой отдачи.

2.  **Этап 2 (Ranking):**
    *   Используется модель **CatBoostClassifier**.
    *   Модель предсказывает вероятность лайка (`predict_proba`) для пар "Пользователь - Пост".
    *   Используемые признаки: 
        *   User features: `gender`, `age`, `country`, `city`, `exp_group`, `os`.
        *   Context features: `month`, `hour`.
        *   Content features: `topic`.
        *   Complex features: `item_similarity_score`.

## Метрики качества
На отложенной выборке (Time-series split):
*   **ROC-AUC:** ~0.84
*   **HitRate@5:** ~ 0.98

## Структура проекта
*   `app.py` — код сервиса (загрузка моделей, обработка запросов).
*   `prepare_df_&_train_model/` — Jupyter Notebook с обучением модели и выгрузкой фичей в БД.
*   `model/` — веса обученной модели CatBoost.

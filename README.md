# Система рекомендаций постов для социальной сети

Проект реализован в рамках обучения на **Karpov.Courses (StartML)**.

Сервис по подбору персональных рекомендаций постов для пользователей социальной сети. Проект прошел полный цикл разработки: от создания baseline-модели и добавления нейросетевых признаков до проведения и анализа A/B-эксперимента для валидации улучшений.

## Задача

Разработать и развернуть REST API сервис, который для каждого пользователя возвращает топ-5 постов, наиболее релевантных его интересам. Проверить гипотезу об улучшении качества рекомендаций с помощью A/B-тестирования.

## Стек технологий

- **Python 3.10**
- **FastAPI, Uvicorn**: Реализация и запуск REST API сервиса.
- **CatBoost**: Ранжирование кандидатов (Ranking).
- **PostgreSQL, SQLAlchemy**: База данных для хранения признаков, текстов и эмбеддингов.
- **Pandas, scikit-learn**: Обработка данных, PCA-преобразование.
- **Transformers (Hugging Face), PyTorch**: Генерация нейросетевых эмбеддингов текста (DistilBERT).
- **SciPy, Statsmodels**: Статистический анализ результатов A/B-теста.

## Архитектура решения

Сервис реализует **A/B-тестирование** для сравнения двух моделей. Пользователи делятся на две группы (`control` и `test`) в соотношении 50/50 с помощью хэш-функции (MD5) от `user_id` с добавлением соли, что обеспечивает стабильное и случайное распределение.

### Модель А (Контрольная группа)

Baseline-модель, использующая классические признаки для ранжирования.

1.  **Отбор кандидатов**: Для простоты в качестве кандидатов берутся все посты.
2.  **Ранжирование (CatBoostClassifier)**: Модель предсказывает вероятность лайка.
    - **User features**: `gender`, `age`, `country`, `city`, `os`.
    - **Context features**: `month`, `hour`.
    - **Content features**: `topic`.
    - **Item-Based feature**: `item_similarity_score` (схожесть поста с ранее лайкнутыми, на основе матрицы со-встречаемости).

### Модель B (Тестовая группа)

Улучшенная модель, обогащенная признаками на основе семантического содержания постов.

1.  **Генерация эмбеддингов**: Тексты постов векторизуются с помощью предобученной модели **DistilBERT** из библиотеки `transformers`.
2.  **Снижение размерности**: Полученные 768-мерные векторы сжимаются до **15 компонент** с помощью метода главных компонент (PCA).
3.  **Ранжирование (CatBoostClassifier)**: В дополнение ко всем признакам из Модели А, в эту модель подаются 15 новых PCA-признаков, описывающих семантику текста.

## A/B Тестирование и Результаты

Был проведен 2-недельный онлайн-эксперимент для сравнения эффективности двух моделей на реальных пользователях. Анализ проводился по трем ключевым продуктовым метрикам.

| Метрика | Контроль (Модель А) | Тест (Модель B) | P-value | Вывод |
| :--- | :---: | :---: | :---: | :--- |
| **User Conversion** (доля юзеров с ≥1 лайком) | 89.13% | **89.82%** | `0.0045` | ✅ **Значимый рост** |
| **Likes per User** (ср. число лайков на юзера) | 3.487 | **3.593** | `0.0000` | ✅ **Значимый рост** |
| **HitRate** (доля показов с ≥1 лайком) | 70.65% | **71.99%** | `0.0000` | ✅ **Значимый рост** |

### **Вывод по эксперименту**

Новая модель (Тестовая группа) показала **статистически значимое улучшение по всем ключевым метрикам вовлеченности**. Прирост HitRate на **1.34 п.п.** является сильным результатом. Модель рекомендована к выкатке на 100% аудитории.

## Запуск

1.  Установить зависимости:
    ```bash
    pip install -r requirements.txt
    ```

2.  Запустить сервис из корневой папки проекта:
    ```bash
    uvicorn rec_system.app:app --reload
    ```
